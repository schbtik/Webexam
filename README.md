Project Description
This project was created as a retake exam assignment.
The following features were implemented using HTML, CSS, and JavaScript:
 
Implemented functionality
- Dynamic greeting text that appears with a typing (self-printing) effect when the page loads.
- Search input field that filters and displays only the page sections matching the entered text.
- Interactive feedback form: when the user hovers over any input or textarea field, its background color and border change.

The project demonstrates basic skills in:
- HTML page structure
- CSS styling and animations
- JavaScript DOM manipulation and event handling
